FORMULARIO



git init

Inicia un nuevo repositorio Git.

git clone

Clona un repositorio existente en tu máquina local.

git add

Añade cambios al área de preparación.

git commit

Guarda los cambios en el repositorio con un mensaje.

git pull

Obtiene cambios del repositorio remoto y los fusiona.

git push

Envía cambios locales al repositorio remoto.

git branch

Muestra y crea ramas.

git merge

Fusiona cambios de una rama a otra.

git log

Muestra el historial de commits.

git status

Muestra el estado actual del repositorio.
